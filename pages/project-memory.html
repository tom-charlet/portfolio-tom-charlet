Les technologies

Memory Game est un jeu simple de Memory réalisé en node.js et grâce aux fonctionnalités des packages ejs et express.

Ces technologies nous ont permis d'échanger des données entre les pages, ainsi nous avons pu récuperer des variables de points, de temps, etc...
